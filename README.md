# discord-medo-e-delirio-bot

## Comandos

### Vírgulas
```
,virgula {query}
,v {query}
```

### Músicas
```
,musica {query}
,m {query}
```

### Parar
```
,parar
,p
```

## Como usar
- Preencha a variável ``TOKEN`` do seu bot no arquivo ``.env``
- Instale o [Docker](https://docs.docker.com/engine/install/)
- Execute ``docker compose up``
